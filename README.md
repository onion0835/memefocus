# MemeFocus

The Curated Hub for AI & Developer Memes - memefocus.com

MemeFocus is a static website for publishing AI culture, developer humor, prompt workflow jokes, and software-team meme collections.

## Cloudflare Pages

Use the default static-site setup:

- Framework preset: None
- Build command: leave empty
- Build output directory: `/`
- Root directory: leave empty

After connecting the GitHub repository, every push to `main` can trigger a new Pages deployment.

## Project Files

- `index.html` - page markup and SEO metadata
- `styles.css` - responsive visual design
- `script.js` - search, filtering, and card rendering
- `assets/hero.png` - generated hero image
- `robots.txt` - crawler directives
- `sitemap.xml` - initial sitemap
